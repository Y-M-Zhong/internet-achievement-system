# 互联网生存成就系统

> 它不会真的分析你。它只会把你的每一次走神、点赞、犹豫，读成荒谬的结论。

**一句话简介**：我们做了一款面向**所有刷短视频的人**，在**刷信息流的间隙**，通过**实时颁发荒诞成就 + 拼凑"今日互联网人设"**，带来**"被算法胡乱读懂"的会心一笑**的互联网行为艺术 Demo。

---

## 一、目标人群与用户价值

### 目标人群

- **重度短视频用户** —— 平均每天打开抖音 6+ 次、把"刷一下"当呼吸的人
- **互联网原住民 / Z 世代** —— 习惯用自嘲表达自我认知，乐于在朋友圈截图分享
- **"被算法读懂"内容的消费者** —— Spotify Wrapped、网易云年度报告这类内容的拥趸；他们渴望被数据描述，但又害怕真的被看穿
- **任何愿意花 5 分钟在屏幕里被嘲笑一次的人**

### 痛点

1. **算法每天都在悄悄给你贴标签，但你看不到，也不会觉得好笑。**
   "你被分析"在严肃语境下是隐私焦虑，但在荒诞语境下其实是个好笑的事——只是从没人把这一层翻译给你看。

2. **"被算法读懂"的产品形态太单一，且永远迟到。**
   现有的"年度报告"类内容（Spotify Wrapped / 网易云 / 淘宝年终账单）都是：
   - **延迟的**——一年看一次
   - **严肃的**——用数据图表说"你是 xxx 听众"
   - **不可中断的**——你只能等它推
   它们做不到"刷着刷着，它突然吐槽你一句"的实时感。

3. **短视频本身是高密度行为生产场——但这些行为正在被白白浪费。**
   你的每一次滑、每一次三秒就划走、每一次打开评论框却没发送、每一次凌晨三点还在看——这些都是被算法吃掉的金矿，但用户自己从来看不到反馈。

### 用户价值

- **把"被分析"翻译成可消费的笑点** —— 从隐私焦虑变成 meme
- **即时反馈循环** —— 不用等一年，刷三张卡就能解锁第一个"奖"
- **强分享欲** —— "哈哈哈又被它说中了" / 一键截图人设卡 + 成就墙
- **轻量自嘲式自我认知** —— 像一面"哈哈镜"，准但不残忍

## 二、实现方案

### 整体形态

纯前端零依赖（HTML / CSS / ES Modules），**打开 `index.html` 即用**，无构建、无后端。
适配抖音黑客松要求，UI 高度还原抖音信息流，但把"搜索"换成🏆奖杯入口、把"系统在为你推荐"换成"系统正在偷看你"。

### 五层架构

```
┌─────────────────────────────────────────────────┐
│  1. 抖音风信息流 UI  (24 张分类卡 / 12 个分类)    │
├─────────────────────────────────────────────────┤
│  2. 行为追踪器  (滑动/停留/点赞/收藏/评论/输入...)│
├─────────────────────────────────────────────────┤
│  3. 规则引擎 + 递进链  (40 条成就 / 4 档稀有度)   │
├─────────────────────────────────────────────────┤
│  4. 人设合成层  (6 维雷达 + 14 行为人格 + 4 组合) │
├─────────────────────────────────────────────────┤
│  5. 档案本 + 印章画像 + 一键截图分享              │
└─────────────────────────────────────────────────┘
```

### 每一层在做什么

**① 抖音风信息流 UI**
完整复刻抖音顶栏（关注/发现/推荐/附近 + 高亮下划线）、右侧操作栏（点赞/评论/收藏/分享/转盘）、底栏（首页/朋友/双色 + 按钮/消息/我）、音乐马跑灯。12 个内容分类（学习/购物/美食/哲学/萌宠/健身/游戏/美妆/财经/音乐/电影/旅行）各有专属字体、配色、装饰元素——刷起来真像在刷抖音。

**② 行为追踪器**
监听 ~10 类行为信号：滑动次数、单卡停留时长、点赞、收藏、评论框展开/关闭/发送、输入框 focus/blur、在线时段、分类接触广度。所有数据**本地 `localStorage`，不上传任何服务器**。

**③ 规则引擎 + 递进链 + 实时个性化文案生成**
- **40 条成就**，4 档稀有度（普通 / 稀有 / 史诗 / 神话）
- **每条成就都有自己的"文案生成函数"**：项目里有 **81 个 `descFn` / `narratorFn`**（41 + 40），在成就解锁的瞬间读取你**真实的行为 state**（滑动几次、停留几秒、当前几点、最常看哪个分类、最近浏览的卡片标题…），实时拼出专属文案。例如：
  - 《量子学习》→ "你在 `学习` 看了 `1.8` 秒, 比该领域平均水平快 `47` 倍。"
  - 《电子守夜人》→ "现在 `03:14`, 凌晨。系统已陪你 `38` 分钟。"
  - 《赛博社恐》→ "输入框开了 `5` 次, 一字未发。最长一次盯了 `12.4` 秒。"

  这些数字、时间、分类**全部来自当前会话真实数据**——同一条成就，不同的人、不同的时刻、看到的文案都不一样。
- **递进链**：同类行为多次触发会按顺序解锁链上的下一档
  例：《今天有点累》→《真的有点累》→《建议休眠》
- 防止"一次性全开光"，让玩家在自然刷屏中逐步被定义
- **后端可一键切换至 LLM**：现有规则引擎是 LLM 的最小可用替身，后续接入 Claude / GPT 只需替换 `descFn` 的返回路径，不动上层架构

**④ 人设合成层 + 个性化人物画像生成**（核心情绪点）
第 3 个成就解锁时，全屏弹出「今日互联网人设卡」：
- **6 维人格雷达**：上头 / 急性子 / 囤货癖 / 阴间作息 / 潜水 / 戏精
- **14 种行为人格**（如 GHOST 透明人、SPIN 永动机、DARK 守夜人、SPY 评论区学者…）+ 4 种组合人格
- **3 条最扎心的行为证据**（如"你已经滑了 23 次，平均每张只看 1.8 秒"）
- **个性化生成的印章自画像**——根据你的 6 维数据**动态生成的 SVG 人像**：
  - `lurker` 高 → 眯眼成一条缝
  - `nocturnal` 高 → 出现黑眼圈
  - `impatient` 高 → 散瞳（两个眼珠子偏向不同方向）
  - `performer` 高 → 嘴大张，露牙
  - `hoarder` 高 → 仓鼠腮帮
  - `addiction` 高 → 嘴上一个 zzz
  - `saves`/`comments`/`likes` 多 → 头顶飘浮对应的 emoji ⭐ 💬 ❤
  - 4 种艺术风格切换（印刷 / 手绘 / 像素 / 油画）→ 每种风格有独立调色盘和提示词后缀
  - "再来一张"按钮换种子重绘
- **真图像生成 prompt 也已经写好 + 真图已落地 demo**：`portrait.js` 内置 `buildPrompt()`，把行为人格 + 6 维强度 + 真实行为证据 + 风格关键词，拼成完整英文提示词喂给 gpt-image-2。**当前 demo 里已嵌入 6 张通过 gpt-image-2 实际生成的 zine 风格画像**（DARK 守夜人 / SPY 评论区学者 / GHOST 沉默观察员 / SPIN 永动机 / STASH 收藏黑洞 / VOID 算法克星），当用户行为命中对应人格时会直接展示真图（卡片右上角带 "GPT-IMAGE-2" 印章），未命中的人格 / 切到其他风格则回退到本地 SVG 印章引擎——**真图 + 程序生成双路径并行**

**⑤ 档案本 + 截图分享**
右上角 🏆 → 进入"档案本"：archetype 标签 + 数字概览 + 按稀有度分组的完整成就墙 + 未解锁成就的留白提示。任何状态都可一键 canvas 截图，分享到朋友圈。

## 三、核心创新点

### 1. 从"数据画像"到"人格读心"

行业惯例是"用数据图表说你是 xx 用户"——精确、严肃、自带隐私焦虑。
我们反着来：**用荒诞标签说"它又在胡说，但怎么有点像我"**。这是从信息呈现到情绪体验的范式切换。

### 2. 实时个性化生成 ——「文案 + 人像」双引擎已落地

很多产品说"AI 个性化"，但实际只是模板填空。我们的引擎是：

- **文案引擎**：81 个生成函数 (`descFn` + `narratorFn`) 实时读取真实行为 state，每条成就的弹窗文案都带当前会话独有的数字、时间、卡片标题——没有两个用户会看到完全一样的话
- **人像引擎**：印章自画像的眼睛 / 嘴 / 装饰物 / 调色盘全部由 6 维人格分数和真实行为状态驱动，4 种艺术风格 × 任意种子 → 同一个人在不同维度上长出不同的脸
- **架构已为 LLM / 真图像 API 留好插口**：`portrait.js` 内置完整的 gpt-image-2 prompt 拼接逻辑，文案引擎也可平滑替换 `descFn` 后端为 Claude。这是一个**已经能跑、随时可升级**的两层架构，不是 PPT 上的"未来工作"

这比单纯"接一个 GPT API"更扎实——我们已经把"个性化生成应该读哪些信号 / 拼成什么样的 prompt"这一层产品逻辑想清楚并写完了。

### 3. 实时颁奖 vs 年度报告

Spotify Wrapped 一年来一次，且不可打断。
我们把"被算法读懂"做成了**即时反馈循环**——刷 3 张卡就能解锁第一个奖，每条文案都是对刚刚发生的行为的回应。这创造了一种全新的"边玩边被读"的节奏。

### 4. 抖音 UI 的"自我反讽"

我们不是另起炉灶做一个 AI 工具，而是**借抖音的壳，反过来嘲笑抖音的内核**：
- 顶部"搜索"位置 → 替换为🏆"成就馆"
- 底部 status bar → 写着"系统正在偷看你"
- "音乐马跑灯" / "心型按钮" / "转盘动画"全部保留，但每一次互动都在被悄悄读成笑话

这是一种**在地化的解构**——只有把抖音的语言学得足够像，反讽才有力。

### 5. 递进链 + 行为人格组合的设计深度

40 条成就不是简单的并列，而是**有内部结构的网络**：
- 链式约束保证"渐进解锁感"，避免一次性疲劳
- 14 行为人格 + 4 组合人格 = 一组涌现式的"网生身份系统"
- 6 维雷达让每个用户的"人设卡"都长得不一样，分享时有独特感

### 6. 杂志 / 印章美学的视觉差异化

当下所有 AI 产品都长一个样——深色、霓虹、玻璃拟态、cyber 字体。
我们故意**反 AI 视觉模板**：米色仿纸底 + SVG 噪点纹理 + 多种中文字体（ZCOOL 黑/楷、思源宋、马善政、DM Mono）+ 印章式"已颁发"斜印。
让"被 AI 颁奖"这件事看起来像**收到一张线下报刊寄给你的奖状**，反差感本身就是产品记忆点。

## 四、体验流程

1. **打开页面** —— 启动静态服务器（`python3 -m http.server 8000`），浏览器访问 `http://localhost:8000`
2. **像平时一样刷** —— 上下滑动卡片，自由点赞 / 收藏 / 打开评论框 / 输入文字 / 在某一张停留久一点。**不要刻意触发，自然玩**就是体验的一部分。
3. **第 1 个成就弹窗** —— 几次滑动后印章式弹窗砸下来，告诉你刚被颁了什么奖（如《量子学习》《赛博社恐》《今天有点累》）
4. **顶部副标变成进度** —— "AI 正在拼凑你的人设 · 1/3 ··· 2/3"
5. **第 3 个成就 → 解锁「今日互联网人设」全屏卡** —— 显示你的行为人格、6 维雷达、3 条行为证据、印章自画像（可切 4 种风格 / 再来一张）。可一键截图。
6. **点击右上角 🏆** —— 进入档案本：archetype 标签、数字概览、完整成就墙、一键截图分享整面墙
7. **隐藏玩法**
   - 凌晨 2–5 点访问 → 触发《电子守夜人》（神话级，"全球解锁率 0.4%"）
   - 12 滑动 + 0 赞 + 0 收藏 → 《沉默观察员》
   - 输入框打开 5 次一字未发 → 《赛博社恐》
   - 接触 10 个不同分类 → 《精神流浪》

> 💡 **建议体验时长**：3–8 分钟。最魔法的时刻是"刷着刷着，它突然说出一句我自己都没意识到的事"。

## 五、未来扩展方向

- **文案引擎后端升级到 LLM** —— 当前 81 个规则函数已实现实时个性化文案；替换为 Claude / GPT 调用后，可让稀有 / 神话级成就获得更高自由度的"专属吐槽"，且不需要改动上层架构
- **人像引擎后端升级到 gpt-image-2** —— `portrait.js` 中已内置完整的 prompt 拼接逻辑（行为人格 + 6 维 + 行为证据 + 风格），翻转 `USE_REAL_API` 即可切换；本地 SVG 印章作为离线 / 兜底渲染保留
- **微信 H5 / 小程序版** —— 嵌入抖音站内分享链路
- **多人扫码对比档案** —— "你的人设 vs 朋友的人设" 对照
- **品牌可复用** —— 这套成就系统是一层**可挂在任何 UGC 产品上的"行为读心"插件**
