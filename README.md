**项目介绍核心要点：**

- **目标人群** —— 重度短视频用户、自嘲式互联网原住民、把"刷手机"当呼吸的人；任何愿意花 5 分钟在屏幕里被嘲笑一次的人。
- **痛点 & 价值** —— 算法每天都在悄悄给你贴标签，但你看不到、也不会觉得好笑。我们把这一层"标签"翻译成**写在脸上的荒诞成就**：你的每一次走神、收藏不看、深夜刷屏、打了删删了再打，都会被读成一个名字奇怪的奖。让"被算法读懂"这件原本有点惊悚的事，变成可以截图发朋友圈的笑点。
- **实现方案** ——
  1. 纯前端零依赖（HTML/CSS/JS + ES Modules），打开 `index.html` 即用
  2. **行为追踪器**：监听滑动 / 停留时长 / 点赞 / 收藏 / 评论展开 / 输入框 focus·blur·send / 在线时段
  3. **规则引擎 + 递进链**：40 条成就 / 4 档稀有度（普通·稀有·史诗·神话）/ 同类行为递进解锁（《今天有点累》→《真的有点累》→《建议休眠》）
  4. **今日互联网人设卡**：第 3 个成就解锁触发，6 维行为雷达（上头·急性子·囤货·阴间·潜水·戏精）+ 14 种行为人格 + 4 种组合 + 印章式 SVG 自画像
  5. **档案本 + 一键截图分享**：奖杯入口 → 成就墙 / archetype 标签 / 数字概览 / canvas 截图分享
- **核心创新点**
  - **从"数据画像"到"人格读心"**：不是冷冰冰的图表，而是会"胡说但说中了"的标签
  - **抖音 UI 反讽**：完整复刻抖音信息流（顶栏 / 右侧操作栏 / 底栏 + 按钮 / 音乐马跑灯 / 转盘），但把"搜索"换成"奖杯"，把"系统在为你推荐"换成"系统正在偷看你"
  - **杂志/印章美学**：在一片赛博 AI 模板的视觉同温层里，用米色仿纸 + 噪点 + 多种中文字体做出"线下报刊给你颁奖"的反差感
  - **递进链 + 行为人格组合**：避免一次性"全成就开光"，让玩家在自然刷屏中逐步被定义

### 体验流程

1. **打开页面** —— 启动一个静态服务器（`python3 -m http.server 8000`），浏览器访问 `http://localhost:8000`。手机框内即是一份完整模拟的抖音信息流。
2. **像平时一样刷** —— 上下滑动卡片，自由点赞 / 收藏 / 打开评论框 / 输入文字（发不发都可以）/ 在某一张停留久一点。**不要刻意触发，自然玩**就是体验的一部分。
3. **第 1 个成就弹窗** —— 几次滑动后会有一张印章式弹窗砸下来（带粒子 + 颁字音效 + 主题色），告诉你刚被颁了什么奖（如《量子学习》《赛博社恐》）。
4. **顶部副标会变成进度** —— "AI 正在拼凑你的人设 · 1/3 ··· 2/3"。
5. **第 3 个成就 → 解锁「今日互联网人设」全屏卡** —— 显示你的行为人格、6 维雷达、3 条最扎心的行为证据、印章自画像（可切 4 种风格：印刷 / 手绘 / 像素 / 油画 / 再来一张）。可一键截图。
6. **点击右上角 🏆** —— 进入档案本：archetype 标签、数字概览、按稀有度分组的完整成就墙、未解锁成就的提示文案、一键截图分享整面墙。
7. **隐藏玩法** ——
   - 凌晨 2–5 点访问可触发「电子守夜人」（神话级，全球解锁率 0.4%）
   - 12 滑动 + 0 赞 + 0 收藏 = 《沉默观察员》
   - 输入框打开 5 次一字未发 = 《赛博社恐》
   - 接触过 10 个不同分类 = 《精神流浪》

> 💡 **建议体验时长**：3–8 分钟。最魔法的时刻是"刷着刷着，它突然说出一句我自己都没意识到的事"。

## 它在做什么

把"刷抖音"变成一场荒诞游戏：

1. 进入页面 → 看到一个完整模拟的短视频信息流（24 张，覆盖学习/购物/美食/哲学/萌宠/健身/游戏/美妆/财经/音乐/电影/旅行 12 个分类）
2. 你的每个微小动作（滑动 / 停留 / 点赞 / 收藏 / 打开评论 / 输入框聚焦）都会被本地行为追踪器记录
3. 追踪器把事件派发给规则引擎，命中规则触发**成就弹窗**
4. 你被读成：《量子学习》《赛博社恐》《深夜哲学家》《看不懂你》……
5. 同类行为多次出现会触发**递进链**：《今天有点累》→《真的有点累》→《建议休眠》
6. 右上角奖杯 → **档案本**：archetype（"你的当下姿态"）+ 数字概览 + 完整成就墙 + 一键截图分享

**核心理念**：不要真分析用户，要让用户觉得"它又在胡说，但怎么有点像我"。

## 数据流

```
用户行为
  └─→ DOM 事件 ──┐
                ↓
       tracker.dispatch(event)
                ↓
       更新 state（swipes/likes/saves/catViews/longStays/cardEnterTs/...）
                ↓
       遍历 ACHIEVEMENTS.check(state, event)
       同 chain 的成就按声明顺序解锁
                ↓
       命中 → emit('unlock')  ──→  popup.enqueue(ach)
                ↓
       localStorage.save()
```

### 递进链

| 链      | 触发           | 普通 → 稀有 → 史诗 → 神话                            |
| ------- | -------------- | ---------------------------------------------- |
| `tired`    | 长时间停留次数  | 《今天有点累》→《真的有点累》→《建议休眠》              |
| `shopping` | 购物分类浏览数  | 《看看就好》→《又看了》→《钱包警告》                    |
| `shy`      | 输入无发送      | 《赛博社恐》→《字打完了又撤》                          |

链内成就只能按顺序解锁，避免一次性"全开"。

## 视觉设计要点

- **不是 cyber-AI 模板**：用了米色仿纸印章、SVG 噪点纹理、多种中文字体（ZCOOL 黑/楷、思源宋体、马善政、DM Mono）
- **类目专属 vibe**：12 个分类有不同的字体/底色/装饰元素（哲学用衬线斜体, 购物有"限时"红标, 电影上下加黑边, 游戏用 mono 体加 ▸ 起头…）
- **档案本风格成就馆**：印章式 tag、米色色块、archetype 标签（"评论区学者"/"电子吸猫党"…）、印章式"已颁发"斜印
- **AI 留白**：narrator 字段不全是"AI 已 XX"，混入旁观/沉默/吐槽多种声音
