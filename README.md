# 互联网生存成就系统

> AI 一边偷偷观察你刷手机，一边给你颁发离谱的成就。
> Steam 成就系统 × 抖音信息流 × 互联网荒诞性 = 想截图分享的 5 秒钟。

抖音黑客松 Demo · 纯 HTML/CSS/JS · 打开 `index.html` 即用，无依赖。

## 它在做什么

把"刷抖音"这个动作变成一场游戏：

1. 用户进入页面 → 看到一个完整模拟的短视频信息流
2. 用户的每一个微小动作（滑动 / 停留 / 点赞 / 收藏 / 打开评论 / 输入框聚焦）都会被本地行为追踪器记录
3. 追踪器把事件派发给规则引擎，命中的规则触发**成就弹窗**
4. 用户被 AI "误读"成各种荒诞身份：《量子学习》《赛博社恐》《深夜哲学家》……
5. 每个成就有稀有度（普通 / 稀有 / 史诗 / 神话）和"全球解锁率"，截图就是社交货币

**核心不是 AI 真的理解用户，而是 AI 和用户一起创造了一个荒诞又好笑的互联网瞬间。**

## 快速开始

```bash
# 无需 build，无需 npm install
open index.html

# 或起个静态服务器（推荐，避免 module CORS 问题）
python3 -m http.server 8000
# 打开 http://localhost:8000
```

> 因为用了 ES Modules，必须通过 `http://` 而不是 `file://` 打开。

## 为什么这个方向适合黑客松

1. **刷到即懂**，0 学习成本
2. **AI 可无限生成**内容，scale 几乎免费
3. **分享欲极强**，"哈哈哈 AI 又在胡说，但怎么有点像我"
4. **开发量小**，能在 hackathon 周末跑完 Demo
5. **品牌可复用**，成就系统能嵌入任何 UGC 产品

## 设计原则

不要真正分析用户，要进行**基于轻行为的荒诞误读**。
用户应该觉得"AI 在胡说，但怎么有点像我"——而不是"AI 监控了我"。

颁奖逻辑要：
- **轻** — 不读隐私数据
- **快** — 几秒之内必然给反馈
- **离谱** — 误读越大反而越好笑
- **像 Steam** — 稀有度 + 解锁率，让用户产生收集欲
